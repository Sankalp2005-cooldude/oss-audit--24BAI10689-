## Description

This project analyzes Python as an open-source software. It covers its origin, licensing, ethical considerations, Linux footprint, and ecosystem. The project also includes practical implementation of shell scripts to demonstrate real-world Linux usage.

## Shell Scripts

### Script 1: System Identity Report

Displays system information such as kernel version, user name, uptime, and Python version.

### Script 2: FOSS Package Inspector

Checks whether Python is installed on the system and displays package details.

### Script 3: Disk and Permission Auditor

Analyzes important system directories and shows their permissions and sizes.

### Script 4: Log File Analyzer

Counts occurrences of a keyword in system log files and displays recent matching entries.

### Script 5: Open Source Manifesto Generator

Generates a personalized open-source manifesto using user input.
